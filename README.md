# Expense Book

日々の出費と予算をカテゴリ毎に記録し、月ごとの支出状況がひと目でわかる家計簿アプリです。

[サンプル](https://yh-expense-book.web.app/) ※スマートフォン推奨

## 機能一覧

- 出費の登録、更新、削除
  - 日付
  - 金額
  - カテゴリ
  - タイトル
- 月別/カテゴリ別予算の登録、更新、削除
- 月別/カテゴリ別の支出状況のグラフ表示
- 日付別の出費一覧表示
